# AWS Production Environment Showcase

This repository showcases a production-style AWS environment project focused on core cloud infrastructure concepts.

## Project Scope

This project is designed to demonstrate:

- Virtual Private Cloud (VPC) design
- Public and private subnet planning
- EC2 deployment
- Security group design
- Load balancing concepts
- Auto scaling concepts
- CloudWatch monitoring
- IAM least privilege thinking

## What This Showcase Contains

- Architecture overview
- Project objectives
- Deployment flow
- Screenshots
- Key decisions and lessons learned

## Status

Planning phase

## Notes

This is a public showcase repository and does not expose sensitive production code or private implementation details.
